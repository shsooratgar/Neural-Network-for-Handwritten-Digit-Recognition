## API Reference

#### sigmoid

```http
  sig(X)
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `x` | `integer` | **Required**. the sigmoid function |

#### sigmoid derivative

```http
  sig_deriv(x)
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `x`      | `integer` | **Required**. derivative of sigmoid |
